## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/PDAM20/API_PDAM.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run Migration
   ```sh
   npx seqeulize db:migrate
   ```
4. Undo Migration
   ```sh
   npx seqeulize db:migrate:undo:all
   ```
5. Running the Program
   ```sh
   node app.js
   ```
6. Test the API
   ```sh
   Import Documentation API to Insomnia
   ```
